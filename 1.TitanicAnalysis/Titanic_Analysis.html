1. Introduction¶

This dataset contains demographics and passenger information from 891 of the 2224 passengers and crew on board the Titanic. The data include variables as follows:

Passengerid: One for each individual
Survived: Whether this individual was survived or not (0 = No; 1 = Yes)
Pclass: Passenger Class (1 = 1st; 2 = 2nd; 3 = 3rd)
Name: Each passenger's name
Sex: Each passenger's sex
Age: Each passenger's age
SibSp: Number of Siblings/Spouses Aboard
Parch: Number of Parents/Children Aboard
Ticket: Each passenger's ticket Number
Fare: Each passenger's passenger Fare
Cabin: Each passenger's cabin
Embarked: Each passenger's port of Embarkation (C = Cherbourg; Q = Queenstown; S = Southampton)

The aim of this project is to see which factor(s) could predict the survival of people on titanic. Therefore, I am going to explore the relationship between people's survival and other variables using correlations for the first step, then select the most related variables to predict people's survival in a logistic regresson model, given that 'Survived' is binary. Meanwhile, the correlation between the possible predictors will also be considered to avoid multicollinearity among predictors. I expect that sex, passengers' class should be strong predictors for survival, for it is like a social norm that in dangerous environments men should protect women, and rich people usually have more priviliges than poor people.

5. Logistic Regression to Predict Survival¶

From the results of m1, the Log-Likelihood of the model is -335.30, p < .001, which means that the model is effective to predict survival on titanic, with Pseudo R Square = 0.3021. However, from the regression coefficients we could see that the p value for Fare and Embarked are larger than .05, which means they are not good predictors. As mentioned above, Fare is negatively correlated to Pclass, which means it probably should not be in the model. And as mentioned above, embarked is probably not a good predictor since there are a lot more people who embarked from Southampton compared to the other two ports, and this intuitive thinking is supported by the model here. So in Model 2, Fare and Embarked are excluded.

6. Conclusions¶

In general, we found that passengers' class and sex are crucial factors to passengers' survival on titanic. In addition, the interaction between passengers' sex and class also have impact on passengers' survival. Overall, female passengers and passengers of higher class were more likely to survive.
